html&css 정리

checkbox 사용하기

checkbox는 아래와 같이 사용함
fieldset 태그나 legend 태그는 사용하지 않아도 무방함.
실제 checkbox 태그는 div 안의 태그인데 태그에 대한 속성은 아래와 같음
type: checkbox 타입 지정
id: label의 for 속성과 매치하기 위해 사용
name: 해당 checkbox를 나타내는 속성
value: checkbox 내의 해당 value를 구분하기 위한 속성(서버에서 해당 속성으로 구분)

text 세로 정렬

div로 감싸는 text 세로 정렬 시 해당 div의 height와 line-height를 일치 시킨다
예를 들어 height가 40px이면 line-height도 40px로 맞춤

그러나 위와 같은 방법 적용 시 해당 div를 리스트 형식으로 동시에 호출하는 등엔 문제가 발생함.

그렇기에 그러한 경우엔 아래와 같은 방법을 적용

위의 방법에서 중요한 2가지는 아래와 같음

  1. 텍스트를 감싸는 블록의 height 값이 명확해야 한다.
  2. 부모 앨리먼트(div)의 css 속성을 display:table; 자식 앨리먼트(p)의 css 속성을 display:table-cell; 로 지정해주면 vertical-align 속성이 적용된다.
    (IE7 이하 버전에서는 display:table, table-cell이 지원되지 않기 때문에 적용되지 않는다.)

span을 div처럼 활용하기

span태그는 inline 속성이므로 width, height 지정 불가함.
그러나 ‘display: inline-block;’를 지정하면 span 태그를 div 태그와 같이
width, height 지정 가능함.

div를 브라우저(화면) 중앙에 배치하기

사용하고자하는 div (id=intro)가 가로 세로 사이즈가 각각 300, 150 이라고 가정하면..
아래와 같이 div 레이어를 좌측 및 상단으로 절반을 밀어낸후, 화면의 50%영역에 배치하면 중앙에 배치된 효과를 얻을 수 있음.

div 가운데 정렬, div 안의 div 가운데 정렬하기

지정한 div 가운데 정렬:
margin: auto; 혹은 margin-left: auto; margin-right: auto; 로 코딩

div 안의 div 가운데 정렬하기
text-align: center; 로 코드
(텍스트 뿐만이 아닌 위와 같이 div도 정렬 가능)

checkbox, radio 버튼 디자인하기

아래와 같이 코딩

div에 background img 넣고 위치 등 조정하기

background-image: url("/img/image.png");
-> div에 삽입할 이미지(그냥 넣으면 같은 이미지가 반복
background-repeat: no-repeat;
-> 반복하는 이미지가 아닌 이미지 하나만 띄움
background-position: 50%;
-> 이미지 중앙 정렬

input type=”file” 을 css로 디자인하기(submit이나 그 외 버튼도 가능)

input type에는 직접 디자인을 씌을 수가 없으므로 아래와 같은 코딩으로 input 태그를 디자인함
아래 html 코드와 같이 작성 후 input[type=file]을 css 투명도 및 위치조정을 통해 안보이게끔 함
또한 input[type=file]의 사이즈 조정을 통해 버튼크기에 맞추어야함
(버튼크기에 맞추지 않으면 버튼 영역 밖에 클릭해도 버튼이 클릭됨)

submit 버튼은 input[type=submit] 으로,
그 외 버튼은 따로 id 선택자 등을 부여하여 아래와 같은 방식으로 디자인 가능하다

클래스명을 이용한 input type 디자인은 아래와 같음(클래스명을 input 앞에 작성할 것)
