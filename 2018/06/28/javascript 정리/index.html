async, await 사용하기

async, await는 프로그램 코드를 동기화 하기 위해 지원하는 예약어인데 이에 대한 사용은 다음과 같음
동기화를 하고자 할 함수에 async 키워드를 함수명 앞에 붙이고, 함수 안에 동기화 할 코드 앞에 await를 붙임
이렇게 하면 await로 지정한 코드가 종료된 이후 다음 코드가 실행됨으로 인해 비동기로 인한 문제점을 막을 수 있음
